# 🛸 Rick and Morty Explorer (Full-Stack)

Projeto desenvolvido como meu **primeiro contato com o Spring Boot**, conectando um **backend em Java** com um **frontend em React**, consumindo dados da **API oficial do Rick and Morty**.

## 🌟 Sobre a aplicação

A aplicação funciona como um **catálogo de personagens da série**, permitindo:

- Navegar por **várias páginas de personagens**
- Visualizar **detalhes do personagem** (origem, espécie, gênero e status)
- Consumir dados **em tempo real da API externa**

## 🚀 Objetivo do estudo

Praticar conceitos iniciais de **Spring Boot**, especialmente:

- Criação de **endpoints**
- **Consumo de APIs externas**

## 📂 Estrutura do projeto

**Backend (Java / Spring)**  
- `Controller` – rotas da API  
- `Service` – consumo da API externa  
- `DTO` – estrutura de dados enviada ao frontend  

**Frontend (React)**  
- `Components` – elementos da interface  
- `Service` – comunicação com o backend
